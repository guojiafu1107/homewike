# 第一阶段：前期准备（约 1-2 个月）

这是整个装修的基石，规划得越细致，后期麻烦越少。

## 一、明确需求与预算

### 1. 确定装修风格

| 风格 | 特点 | 适合人群 |
| :--- | :--- | :--- |
| **现代简约** | 线条简洁、功能性强、色彩单一 | 追求实用的年轻人 |
| **轻奢风** | 精致优雅、金属元素、质感高级 | 追求品质的中产 |
| **新中式** | 传统元素、东方韵味、沉稳大气 | 喜欢传统文化 |
| **北欧风** | 自然温馨、原木色、明亮通透 | 小户型首选 |
| **日式风** | 极简自然、收纳至上、禅意空间 | 喜欢整洁的人 |
| **美式风** | 复古怀旧、舒适自由、色彩丰富 | 大户型、别墅 |

### 2. 规划功能布局

根据家庭成员和生活习惯，规划每个空间：

**居住需求清单**：
- [ ] 常住人口有几人？
- [ ] 是否有老人/小孩？（影响安全设计）
- [ ] 是否养宠物？
- [ ] 是否经常在家做饭？
- [ ] 是否需要独立书房/衣帽间？
- [ ] 是否需要大量收纳空间？
- [ ] 是否需要智能家居？

### 3. 设定预算

**预算分配黄金比**：

| 项目 | 占比 | 说明 |
| :--- | :--- | :--- |
| **硬装施工** | 40% | 拆除、水电、泥木、油漆 |
| **主材** | 30% | 瓷砖、地板、卫浴、门、橱柜 |
| **软装电器** | 25% | 家具、家电、窗帘、装饰 |
| **备用金** | 5% | 应对增项和突发情况 |

::: warning 重要提醒
务必预留 **10%-15%** 作为备用金！装修过程中难免有增项，预留备用金能应对突发情况。
:::

**预算表示例（以 100㎡ 中档装修为例）**：

| 项目 | 金额 | 包含内容 |
| :--- | :--- | :--- |
| 硬装施工 | 6-8 万 | 人工+辅材 |
| 主材 | 4-6 万 | 瓷砖、地板、卫浴等 |
| 软装电器 | 4-5 万 | 家具、家电 |
| 备用金 | 1-2 万 | 应急资金 |
| **总计** | **15-20 万** | - |

## 二、选择装修方式与团队

### 三种装修方式对比

| 方式 | 说明 | 优点 | 缺点 | 适合人群 |
| :--- | :--- | :--- | :--- | :--- |
| **清包** | 自己设计、买所有材料，施工队只施工 | 预算完全可控 | 极其耗时，需懂行 | 时间充裕且懂行 |
| **半包** | 装修公司负责设计、辅材和施工，自己买主材 | 性价比高，主材可控 | 需花时间跑市场 | **大多数家庭** |
| **全包** | 装修公司负责一切 | 最省心 | 费用高，材料不可控 | 时间少的业主 |

::: tip 推荐
**半包**是大多数家庭的最佳选择：
- 主材自己把控质量和环保
- 辅料由装修公司负责
- 总价相对可控
:::

### 签订合同注意事项

1. **对比 3-5 家**正规公司，查看资质和过往案例
2. **合同必须明确**：
   - 材料品牌、型号、规格
   - 工期起止时间
   - 付款方式（建议 3-3-3-1）
   - 违约责任
   - 质保期限

## 三、验房清单（精装房/二手房）

### 必备验房工具

| 工具 | 用途 | 价格 |
| :--- | :--- | :--- |
| 空鼓锤 | 检查墙面/地面空鼓 | 10-20 元 |
| 卷尺 | 测量尺寸 | 15-30 元 |
| 水平仪 | 检查墙面平整度 | 30-50 元 |
| 验电器 | 检测电路 | 20-40 元 |

### 验房检查项目

- **门窗**：密封条完好，锁具顺滑，玻璃无划痕
- **墙面**：空鼓率不超过 5%，无裂缝
- **地面**：平整度误差 < 3mm，无空鼓
- **水电**：水压测试（0.8MPa 保压 30 分钟），电路回路正常
- **防水**：卫生间闭水试验通过
