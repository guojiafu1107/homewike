# 第三阶段：主材安装（约 15-30 天）

硬装基础完成后，各类主材开始进场安装，**顺序很重要**！

## 安装顺序总览

```
┌─────────────────────────────────────────┐
│  1. 厨卫吊顶 → 2. 橱柜安装 → 3. 木门安装  │
│     ↓                                    │
│  4. 木地板 → 5. 卫浴洁具 → 6. 开关灯具    │
│     ↓                                    │
│  7. 定制柜体 → 8. 竣工验收               │
└─────────────────────────────────────────┘
```

---

## 一、厨卫吊顶安装

### 安装内容

- 铝扣板/石膏板/蜂窝大板吊顶
- 浴霸安装
- 排气扇安装
- 集成灯安装

### 注意事项

1. **提前购买**：浴霸、排气扇需提前确定型号
2. **预留检修口**：方便日后维修
3. **止逆阀安装**：排气扇必须装止逆阀，防止倒灌

### 材料选择

| 材料 | 优点 | 缺点 | 适用 |
| :--- | :--- | :--- | :--- |
| 铝扣板 | 防潮、易拆卸、便宜 | 美观度一般 | 厨房、卫生间 |
| 蜂窝大板 | 美观、整体性好 | 价格较高 | 大户型厨卫 |
| 石膏板 | 造型丰富、美观 | 防潮性差 | 客餐厅、干区 |

---

## 二、橱柜安装

### 安装内容

- 地柜安装
- 吊柜安装
- 台面安装
- 水槽、龙头安装
- 五金件安装（拉篮、抽屉等）

### 验收要点

| 检查项 | 标准 |
| :--- | :--- |
| 柜体水平 | 误差 < 2mm |
| 柜门对齐 | 缝隙均匀，开关顺畅 |
| 台面接缝 | 打磨平整，无明显缝隙 |
| 水槽安装 | 打胶饱满，无渗漏 |
| 五金件 | 抽拉顺滑，无卡顿 |

### 注意事项

1. **先装橱柜再装门**：避免磕碰
2. **台面开孔**：现场开孔注意防尘
3. **打胶处理**：与墙面、地面交接处打胶密封

---

## 三、木门/垭口套安装

### 安装顺序

**先装门 → 再铺地板 → 最后踢脚线**

### 门的类型选择

| 类型 | 特点 | 适用 |
| :--- | :--- | :--- |
| 实木门 | 环保、质感好 | 卧室 |
| 实木复合门 | 性价比高、稳定 | 卧室、书房 |
| 模压门 | 便宜、造型多 | 出租房 |
| 推拉门 | 节省空间 | 厨房、阳台 |
| 折叠门 | 开启面积大 | 小户型隔断 |

### 验收要点

- [ ] 门扇开关顺畅，无卡顿
- [ ] 门锁安装牢固，开关灵活
- [ ] 门套垂直，与墙面贴合紧密
- [ ] 合页安装牢固，数量足够（一般 3 个）
- [ ] 密封条安装到位

---

## 四、木地板安装

### 安装前准备

- 地面找平已完成
- 地面干燥（含水率 < 12%）
- 地面清洁干净

### 地板类型对比

| 类型 | 环保性 | 脚感 | 价格 | 保养 |
| :--- | :--- | :--- | :--- | :--- |
| 实木地板 | ★★★★★ | ★★★★★ | 高 | 需打蜡保养 |
| 实木复合 | ★★★★☆ | ★★★★☆ | 中 | 简单打理 |
| 强化复合 | ★★★☆☆ | ★★★☆☆ | 低 | 简单打理 |

### 安装方式

| 方式 | 特点 | 适用 |
| :--- | :--- | :--- |
| 龙骨+地板 | 脚感好，有架空层 | 实木地板 |
| 悬浮铺装 | 快捷方便，无胶环保 | 复合地板 |
| 胶粘铺装 | 稳固，但用胶 | 特殊场合 |

### 验收要点

- [ ] 表面平整，无起拱
- [ ] 接缝严密，无高低差
- [ ] 踢脚线安装牢固，与墙面贴合
- [ ] 门口收边处理得当
- [ ] 走动无异响

---

## 五、卫浴洁具安装

### 安装内容

| 洁具 | 注意事项 |
| :--- | :--- |
| **马桶** | 检查下水管距墙距离（305mm/400mm），法兰安装到位 |
| **浴室柜** | 镜柜高度（离地 130-140cm），柜体水平 |
| **花洒** | 混水阀高度（离地 100-110cm），左热右冷 |
| **浴缸** | 提前进场，做好保护 |
| **五金挂件** | 位置合理，安装牢固 |

### 验收要点

- [ ] 马桶冲水顺畅，无渗漏
- [ ] 浴室柜无晃动，下水通畅
- [ ] 花洒出水均匀，无渗漏
- [ ] 所有下水通畅，无返味

---

## 六、开关插座、灯具安装

### 安装前准备

- 全屋通电测试
- 检查每个回路是否正常
- 准备临时照明

### 开关插座验收

| 检查项 | 标准 |
| :--- | :--- |
| 安装高度 | 开关 130cm，插座 30cm（常规） |
| 安装水平 | 相邻插座误差 < 1mm |
| 通电测试 | 每个插座通电正常 |
| 接地测试 | 接地可靠 |
| 开关控制 | 控制正确（相线控制） |

### 灯具安装

- [ ] 主灯安装牢固，无晃动
- [ ] 筒灯、射灯位置正确
- [ ] 灯带效果均匀
- [ ] 智能灯具连接正常

---

## 七、全屋定制柜体安装

### 安装内容

- 衣柜
- 电视柜
- 餐边柜
- 玄关柜
- 阳台柜

### 验收要点

| 检查项 | 标准 |
| :--- | :--- |
| 柜体水平垂直 | 误差 < 2mm |
| 柜门对齐 | 缝隙均匀 ≤ 2mm |
| 开合顺畅 | 无卡顿、无异响 |
| 五金件 | 品牌与合同一致，功能正常 |
| 封边 | 无翘边、无脱胶 |
| 内部结构 | 与设计方案一致 |

### 注意事项

1. **现场保护**：安装时注意保护地板和墙面
2. **卫生清理**：安装后及时清理包装垃圾
3. **调试检查**：所有抽屉、门板都要调试到位

---

## 安装阶段检查清单

### 厨卫区

- [ ] 吊顶平整，电器安装牢固
- [ ] 橱柜水平，台面无缝
- [ ] 水槽下水通畅，无渗漏
- [ ] 卫浴洁具安装牢固
- [ ] 五金挂件位置合理

### 客卧区

- [ ] 木门开关顺畅，门锁正常
- [ ] 木地板平整，无异响
- [ ] 踢脚线安装牢固
- [ ] 开关插座通电正常
- [ ] 灯具照明效果满意

### 收纳区

- [ ] 定制柜体安装到位
- [ ] 柜门对齐，开合顺畅
- [ ] 五金件功能正常
- [ ] 内部结构符合设计

::: tip 温馨提示
主材安装阶段要协调好各厂家进场顺序，避免互相干扰。建议让装修公司统一协调，或由工长安排时间表。
:::
