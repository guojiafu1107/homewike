# 🏗️ 材料库

家装材料分为主材和辅材两大类。主材决定装修效果和风格，辅材决定工程质量。

## 主材 (主要装饰材料)

主材是决定家装风格和预算的核心部分，通常在装修后期安装。

### 地面材料

| 材料 | 特点 | 适用空间 |
| :--- | :--- | :--- |
| [瓷砖](./floor/tile) | 防水、耐擦洗、易清洁 | 厨房、卫生间、客厅 |
| 木地板 | 脚感舒适、温馨自然 | 卧室、书房 |
| 石材 | 高档、耐用 | 窗台、台面、玄关 |

**木地板分类**：
- **实木地板**：天然木材，环保性好，价格较高，需保养
- **实木复合地板**：稳定性好，性价比高
- **强化复合地板**：耐磨、便宜，但脚感稍差

**石材分类**：
- **大理石**：天然纹理，高档，需定期养护
- **石英石**：人造石材，硬度高，耐污，适合做台面
- **岩板**：新型材料，超薄超大，性能优异

### 墙面材料

| 材料 | 特点 | 适用空间 |
| :--- | :--- | :--- |
| [涂料/油漆](./wall/paint) | 施工简单、颜色丰富、易翻新 | 全屋墙面 |
| 壁纸/墙布 | 图案丰富、装饰性强 | 客厅、卧室背景墙 |
| 护墙板 | 高档、保护墙面 | 客厅、卧室 |

---

### 厨卫设施

| 设施 | 功能 | 选购要点 |
| :--- | :--- | :--- |
| **橱柜** | 厨房收纳和操作核心 | 柜体材质、五金件、台面 |
| **洁具** | 马桶、洗手盆、浴缸 | 釉面质量、冲水方式、节水 |
| **水槽** | 厨房清洗中心 | 材质（不锈钢/石英石）、尺寸、安装方式 |

**橱柜要点**：
- 柜体：多层板 > 颗粒板 > 密度板
- 台面：石英石 > 人造石 > 不锈钢
- 五金：百隆、海蒂诗等进口品牌更耐用

---

### 门窗及吊顶

| 类别 | 包含 | 选购要点 |
| :--- | :--- | :--- |
| **门** | 入户门、室内门、推拉门 | 材质、隔音、五金 |
| **窗户** | 断桥铝窗、系统窗 | 型材厚度、玻璃、五金 |
| **吊顶** | 铝扣板、石膏板 | 防潮、防火、造型 |

**窗户关键参数**：
- **型材**：断桥铝，壁厚 ≥ 1.4mm
- **玻璃**：双层中空钢化玻璃
- **五金**：品牌五金，保证开启顺畅

### 电器与五金

| 类别 | 包含 | 说明 |
| :--- | :--- | :--- |
| **灯具** | 主灯、筒灯、射灯、灯带 | 照明 + 装饰作用 |
| **开关插座** | 各类开关、插座、弱电面板 | 安全、便利、美观 |
| **厨卫电器** | 热水器、烟机灶具、洗碗机 | 按功能需求选择 |
| **散热器/地暖** | 暖气片、地暖系统 | 北方必备 |

## 辅材 (辅助基础材料)

辅材虽然不直接外露，但直接关系到装修工程的质量和耐久性。

### 基础建材

| 材料 | 用途 | 选购要点 |
| :--- | :--- | :--- |
| **水泥** | 砌墙、找平、贴砖 | 标号 32.5 或 42.5，注意保质期 |
| **沙子** | 配合水泥使用 | 河沙优于海沙，含泥量 < 3% |
| **砖头** | 砌墙、包管 | 红砖、轻质砖 |

---

### 水电管线

::: warning 重要提示
水电是家庭的"血管"和"神经"，安全至关重要！必须使用合格产品。
:::

| 材料 | 用途 | 选购要点 |
| :--- | :--- | :--- |
| **电线** | 电路传输 | 国标铜芯线，照明 1.5²，插座 2.5²，空调 4² |
| **PPR 水管** | 给水管道 | 壁厚 ≥ 3.4mm，抗菌型更好 |
| **PVC 线管** | 电线保护 | 强电红管，弱电蓝管 |
| **PVC 排水管** | 下水管道 | 50mm、75mm、110mm 等规格 |

**推荐品牌**：
- 电线：熊猫、起帆、正泰
- 水管：伟星、日丰、金德

---

### 板材与龙骨

| 材料 | 用途 | 选购要点 |
| :--- | :--- | :--- |
| **石膏板** | 吊顶、隔墙 | 泰山、龙牌，厚度 9.5mm/12mm |
| **木工板** | 基层、造型 | E0/E1 级环保标准 |
| **轻钢龙骨** | 吊顶骨架 | 镀锌处理，防锈 |
| **木龙骨** | 造型基础 | 需做防火、防腐处理 |

::: tip 建议
吊顶必须使用轻钢龙骨，禁止使用木龙骨（易变形、易燃）。
:::

### 辅料与胶漆

| 材料 | 用途 | 选购要点 |
| :--- | :--- | :--- |
| **腻子粉** | 墙面找平 | 耐水腻子（N 型）优于普通腻子 |
| **防水涂料** | 卫生间、阳台防水 | 雨虹、德高，墙面刚性，地面柔性 |
| **白乳胶** | 木工粘接 | 环保型，看检测报告 |
| **玻璃胶** | 收边密封 | 防霉型，中性胶 |
| **瓷砖胶** | 瓷砖铺贴 | 代替水泥砂浆，粘结力更强 |

### 小五金件

虽小但重要的配件：

| 五金件 | 用途 | 选购建议 |
| :--- | :--- | :--- |
| **地漏** | 排水 + 防臭 | 潜水艇、九牧，选深水封或 T 型 |
| **角阀** | 控制水源 | 铜材质，陶瓷阀芯 |
| **合页/滑轨** | 门窗开启 | 不锈钢材质，带缓冲更好 |
| **拉手/把手** | 开关操作 | 与装修风格匹配 |
| **挂钩/置物架** | 收纳 | 免打孔或打孔安装 |

## 主材 vs 辅材预算占比

| 项目 | 占比 | 说明 |
| :--- | :--- | :--- |
| **主材** | 30-40% | 瓷砖、地板、卫浴、门、橱柜等 |
| **辅材** | 15-20% | 水泥、电线、水管、石膏板等 |
| **人工费** | 25-30% | 各工种施工费用 |
| **家具家电** | 20-25% | 可后期逐步添置 |

::: tip 省钱建议
主材档次差异大，建议重点投入高频使用和显眼的材料（如客厅瓷砖、橱柜），次要空间可选性价比产品。
:::
