# 瓷砖选购百科

瓷砖是家装中用量最大的地面材料之一，常用于厨房、卫生间、客厅，具有防水、耐擦洗的优点。

## 核心参数解读

| 参数 | 选购建议 | 说明 |
| :--- | :--- | :--- |
| **吸水率** | 全瓷砖 < 0.5%（硬度高）；半瓷砖 > 0.5%（需泡水） | 吸水率越低，密度越高 |
| **耐磨度** | 家用选 PEI 3-4 级即可 | PEI 5 级适合商业场所 |
| **防滑性** | 卫生间 R 值 > 12 | R 值越大越防滑 |
| **平整度** | 翘曲度 < 0.5% | 影响铺贴效果 |
| **放射性** | 选 A 类标准 | 室内必须使用 A 类 |

### 瓷砖类型对比

| 类型 | 吸水率 | 特点 | 适用空间 |
| :--- | :--- | :--- | :--- |
| 抛光砖 | < 0.5% | 光亮耐磨，易滑 | 客厅、卧室 |
| 抛釉砖 | < 0.5% | 花色丰富，亮度高 | 客厅、背景墙 |
| 通体砖 | < 0.5% | 硬度高，耐磨 | 客厅、阳台 |
| 仿古砖 | 0.5-3% | 防滑好，复古风 | 阳台、露台 |
| 瓷片 | 3-10% | 价格低，需泡水 | 厨卫墙面 |

## 避坑指南

### 1. 产地识别

- **广东砖**：质量普遍优于其他产区，密度高、釉面好
- **山东砖**：价格便宜，但密度和釉面一般
- **福建砖**：仿古砖有特色

**识别方法**：看包装箱生产地址，不要只看营销地址。

### 2. 色差控制

- 同一批次购买，不同批次会有色差
- 建议多买 5%-8% 作为备货，退补方便
- 收货时核对批次号是否一致

### 3. 厚度与重量

| 规格 | 正常厚度 | 正常重量（单片） |
| :--- | :--- | :--- |
| 800×800mm | 9-12mm | 16-18kg |
| 600×600mm | 9-10mm | 8-10kg |
| 300×600mm | 8-9mm | 3-4kg |

::: warning 警惕
厚度明显偏薄、重量偏轻的瓷砖，密度可能不达标。
:::

### 4. 常见套路

1. **以次充好**：样品与送货不同，一定要验货
2. **虚标规格**：实际尺寸与标称不符
3. **冒充广东砖**：山东砖贴广东牌

## 选购实操步骤

### 第一步：确定规格

| 空间 | 推荐规格 | 说明 |
| :--- | :--- | :--- |
| 客厅 | 800×800mm / 750×1500mm | 大气美观 |
| 卧室 | 800×800mm / 600×1200mm | 温馨舒适 |
| 厨卫地面 | 300×300mm / 400×400mm | 方便找坡 |
| 厨卫墙面 | 300×600mm / 400×800mm | 竖贴显高 |

### 第二步：现场测试

1. **看**：表面无针孔、无色斑、无色差
2. **听**：敲击声音清脆，说明密度高
3. **测**：两块砖面对面贴合，检查平整度
4. **划**：钥匙划釉面，检查耐磨性
5. **量**：用卷尺量对角线，检查方正度

### 第三步：计算用量

```
用量 = 铺贴面积 ÷ 单片面积 × 1.08（损耗）
```

**示例**：客厅 30㎡，用 800×800mm 砖
- 单片面积 = 0.8 × 0.8 = 0.64㎡
- 理论用量 = 30 ÷ 0.64 = 47 片
- 实际用量 = 47 × 1.08 = 51 片
