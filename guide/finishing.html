第四阶段：软装与收尾（约 15-30 天）

装修进入最后阶段，为新家注入温度和个性。

一、开荒保洁

保洁时机

在所有硬装和安装工作结束后，进行一次彻底的全面清洁。

清洁范围

区域清洁内容
玻璃窗户玻璃、玻璃门、镜子
墙面开关面板、踢脚线、墙面浮灰
地面地板、瓷砖深度清洁，去除胶渍、涂料
厨卫橱柜内外、洁具、五金件
定制柜柜体内外、抽屉轨道
灯具灯罩、灯体表面

清洁方式选择

方式优点缺点建议
专业保洁专业工具，效率高费用 500-1500 元推荐
自己动手省钱费时费力，难彻底有时间和体力

专业保洁注意事项

  1. 提前预约：装修快结束时预约
  2. 现场监督：防止野蛮清洁损坏物品
  3. 验收标准：按区域逐项验收
  4. 保护成品：清洁后做好保护，等待家具进场

二、软装进场

软装是营造家的氛围的关键，让空间充满生活气息。

软装进场顺序

1. 窗帘 → 2. 成品家具 → 3. 家电 → 4. 装饰品

1. 窗帘安装

安装时机：家具进场前

空间窗帘类型选购要点
客厅纱帘+布帘遮光率 50-70%，注重美观
卧室遮光帘遮光率 80% 以上，保证睡眠
厨房百叶帘/卷帘易清洁，防水防油
卫生间防水卷帘隐私性好，防潮

窗帘计算：

  • 宽度 = 窗宽 × 2（褶皱倍数）
  • 高度 = 窗高 + 30cm（离地高度调整）

2. 成品家具进场

主要家具清单：

空间家具
客厅沙发、茶几、电视柜、边几
餐厅餐桌、餐椅、餐边柜
卧室床、床头柜、梳妆台/书桌、衣柜（成品）
书房书桌、书椅、书架

进场注意事项

  1. 提前测量：电梯、门框尺寸，确保能搬进
  2. 保护地面：搬运时铺好保护垫
  3. 检查质量：现场检查有无破损、色差
  4. 摆放定位：按设计图纸摆放

3. 家电进场

进场顺序：

顺序家电注意事项
1冰箱静置 2 小时后通电，预留散热空间
2洗衣机检查排水、进水是否通畅
3空调专业安装，抽真空，测试制冷
4电视壁挂需专业安装，检查屏幕
5小家电按需摆放

4. 装饰品布置

装饰品种类：

类型作用举例
装饰画点缀墙面挂画、照片墙
绿植增添生机龟背竹、琴叶榕、绿萝
摆件个性展示花瓶、雕塑、收藏品
纺织品温馨氛围抱枕、地毯、桌布
灯具氛围营造台灯、落地灯、氛围灯

搭配原则

  • 色彩协调，与整体风格统一
  • 层次分明，高低错落
  • 适度留白，避免过度堆砌

三、竣工验收

验收项目清单

墙面验收

  • 墙面平整，无裂缝、无脱落
  • 涂料色泽均匀，无色差
  • 阴阳角平直

地面验收

  • 瓷砖无空鼓（空鼓率 < 5%）
  • 地板平整，无起拱、无异响
  • 踢脚线安装牢固

水电验收

  • 所有插座通电正常
  • 开关控制正确
  • 灯具照明正常
  • 给排水通畅，无渗漏

门窗验收

  • 门窗开关顺畅
  • 密封条安装到位
  • 锁具使用正常

五金件验收

  • 水龙头无渗漏
  • 门把手、合页灵活
  • 洁具安装牢固

定制柜体验收

  • 柜门对齐，开合顺畅
  • 抽屉抽拉顺滑
  • 五金件功能正常

验收工具

工具用途
空鼓锤检查瓷砖空鼓
水平尺检查平整度
卷尺测量尺寸
验电器检测电路
手电筒检查细节

问题整改

  1. 记录问题：拍照/录像，详细记录
  2. 书面确认：要求装修公司签字确认整改事项
  3. 限期整改：约定整改期限
  4. 复验确认：整改后再次验收

四、入住前准备

必备物品采购

类别物品
清洁用品扫把、拖把、垃圾桶、垃圾袋
厨房用品锅具、餐具、刀具、砧板
卫浴用品毛巾、浴巾、洗漱用品
床上用品床垫、床单、被褥、枕头
日常用品衣架、拖鞋、工具箱

智能设备设置

  • Wi-Fi 网络设置
  • 智能门锁设置
  • 智能家居设备连接
  • 监控设备安装

重要资料整理

  • 装修合同及附件
  • 各类发票、保修卡
  • 水电走向图
  • 材料检测报告
  • 质保承诺书
温馨提示：竣工验收通过后，记得向装修公司索要质保卡和水电走向图，这些都是日后维修的重要依据！
